# Drupal

Contains Docker images for testing Drupal.

## Drupal
### Included packages

- git
- libzip-dev
- zip
- libpng-dev
- mariadb-client
- chromium-driver
- gh (Github CLI)

### PHP extensions

- php-gd
- php-pdo
- php-zip
- php-bcmath

### PHP libraries

- composer

## Robotframework

Docker image used to test Drupal using Robotframework.
